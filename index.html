From AI to AO: Predicting How Points End in Tennis

Predicting How Points End in Tennis was a competition hosted on CrowdAnalytix. Since this was my first official data science competition, I spent a lot of time learning and exploring different algorithms to get a feel about how each algorithm worked. The goal of the compeittion was: given certain features about how a point developed (such as how fast the last shot was or how far the players was away from the net), the goal was to predict how the point ended: winner, unforced error, or forced error. The model I settled on was XGBoost, which is a super popular algorithm among Kagglers. The code can be found here.
