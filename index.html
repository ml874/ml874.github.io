From AI to AO: Predicting How Points End in Tennis

Github | Final XGBoost Model | Neural Network

Predicting How Points End in Tennis was a competition hosted on CrowdAnalytix. This was also my first data science competition, so I spent a lot of time learning and exploring different algorithms to get a feel about how each algorithm worked. The goal of the competition was to predict how the point ended: winner, unforced error, or forced error. The model I settled on was XGBoost, which is the go-to algorithm among Kagglers.

Crypto Market Sentiment Analysis/Reddit Metrics Data Scraper

Reddit Metrics Scraper | Live Social Stats

As a trader, it is important to have "signals", or indicators, to determine if a trade should be made. Given that cryptocurrencies are relatively new, there aren't many trading signals to rely upon. For example, we could look at Github commits to see if a crypto is worth putting money in, or look at the market sentiment for various cryptos. This project focuses on the latter example- sentiment. But how do we determine market sentiment for cryptos?

Reddit seems to be the main source of information for "investors". Naturally, following various crytpo subreddits is a good start to guage market sentiment. However, Reddit doesn't readily supply such information. Luckily, Reddit Metrics, a separate site, does. It holds subreddit data, such as ubscriber Growth, Subscriber Count, Milestones, etc...

So this project was dedicated to pulling that data from Reddit Metrics. One script also utilizes CryptoCompare's API to pull general social statistics from sites like Facebook, Githhub, and Twitter.
